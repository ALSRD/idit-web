## 📋 Descripción del Proyecto

El sitio web del IDIT es la presencia digital oficial del instituto. Está construido con tecnologías web estándar (HTML, CSS, JavaScript) y desplegado automáticamente mediante GitHub Pages. El objetivo es ofrecer una plataforma informativa y accesible para estudiantes, docentes y el público en general.

### Funcionalidades actuales
- Página principal informativa
- SEO avanzado con JSON-LD (EducationalOrganization + WebSite)
- Optimizado para motores de búsqueda
- Responsive design

### Funcionalidades por desarrollar
- Sección de cursos y programas académicos
- Formulario de inscripción
- Portal de noticias e eventos
- Sección de docentes
- Galería multimedia
- Versión en inglés (i18n)

## 🚀 Cómo empezar (Setup local)

```bash
# 1. Haz fork del repositorio desde GitHub

# 2. Clona tu fork
git clone https://github.com/TU-USUARIO/idit-web.git

# 3. Entra al directorio
cd idit-web

# 4. Abre el proyecto en tu editor
code .

# 5. Abre index.html en tu navegador para ver el sitio
# Puedes usar la extensión Live Server de VSCode
```

No se requieren dependencias ni instalaciones adicionales. El proyecto es puro HTML/CSS/JS.

## 🤝 Cómo Contribuir

¡Las contribuciones son bienvenidas! Consulta el archivo **[CONTRIBUTING.md](CONTRIBUTING.md)** para ver el proceso completo.

Resumen rápido:
1. Haz **Fork** del proyecto
2. Crea tu rama: `git checkout -b feature/nombre-de-tu-mejora`
3. Realiza tus cambios y haz commit: `git commit -m 'feat: descripción del cambio'`
4. Sube tu rama: `git push origin feature/nombre-de-tu-mejora`
5. Abre un **Pull Request** hacia `main`

### Tipos de contribuciones bienvenidas
- 🐛 Corrección de bugs
- 🎨 Mejoras de diseño y UI/UX
- ♿ Mejoras de accesibilidad
- 📱 Mejoras de responsividad mobile
- ✨ Nuevas secciones o funcionalidades
- 🌐 Traducciones
- 📝 Mejoras en documentación
- ⚡ Optimizaciones de rendimiento / SEO
